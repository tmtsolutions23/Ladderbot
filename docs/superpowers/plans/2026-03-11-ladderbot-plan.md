# LadderBot Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a Python tool that identifies +EV 2-leg cross-game parlays for NBA/NHL, tracks a $10→$1000 ladder, and provides CLI + web dashboard + Discord alerts.

**Architecture:** Modular Python package with data layer (API clients + cache), ML models (logistic regression + Dixon-Coles), parlay optimizer, ladder state machine, FastAPI web dashboard, and Discord webhook alerts. SQLite for all persistent data.

**Tech Stack:** Python 3.11+, FastAPI, uvicorn, httpx, scikit-learn, scipy, numpy, pandas, nba_api, Chart.js, Tailwind CSS
